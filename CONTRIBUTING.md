## How to Contribute

### Reporting Bugs
- Check if the issue already exists in the [Issues](https://github.com/Ahmed-Elshitehi/My-Chat-App/issues) section.
- If not, create a new issue with:
  - A descriptive title
  - Steps to reproduce the problem
  - Screenshots (if applicable)

### Suggesting Features
- Open a new issue labeled as `enhancement` or `feature request`.
- Describe the problem you want solved and your proposed solution.

### Submitting Code
1. Fork the repository.
2. Create a new branch for your feature or fix:
   ```bash
        git checkout -b feature/your-feature-name
   ```
3. Make your changes and commit them:
    ```bash
        git commit -m "Add [feature/fix description]"
    ```
4. Push your branch to your fork:
    ```bash
        git push origin feature/your-feature-name
    ```

5. Open a Pull Request on the main repository with a clear description of your changes.

### Code Guidelines
- Follow existing code style and formatting.
- Write clear, meaningful commit messages.
- Include tests for new features or bug fixes if possible.

### Communication & Conduct
- Be respectful and constructive in discussions.
- Discuss major changes before implementing them.
- Collaborate openly to improve the project.
